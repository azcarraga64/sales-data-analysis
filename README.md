# 📊 Sales Data Analysis

This project analyzes retail sales data to identify trends and business insights.

## Tools Used

- Python
- Pandas
- Matplotlib

## Key Insights

- Technology products generate the highest revenue.
- The West region shows the strongest sales performance.
- A small group of products contributes to a large share of revenue.

## Goal

Help businesses understand sales performance and identify opportunities for growth.

## Dataset

Superstore Sales Dataset
